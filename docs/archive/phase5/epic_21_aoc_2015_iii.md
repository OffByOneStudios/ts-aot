# Epic 21: AoC 2015 Solutions III

**Status:** Planned
**Parent:** [Phase 5 Meta Epic](./meta_epic.md)

## Overview
Solve Days 7, 8, and 9 of Advent of Code 2015 to validate Phase 5 features.

## Milestones

### Milestone 21.1: Day 7 - Some Assembly Required
- **Input:** Circuit instructions (AND, OR, LSHIFT, NOT).
- **Logic:** Bitwise operations, signal propagation.
- **Challenge:** Memoization/Caching of signal values.
- **Feature Usage:** `switch` (for parsing ops), `Map` (for wires).

### Milestone 21.2: Day 8 - Matchsticks
- **Input:** String literals with escapes.
- **Logic:** Calculate code length vs memory length.
- **Challenge:** Handling escape sequences (`\\`, `\"`, `\xNN`).
- **Feature Usage:** Template literals (maybe), String processing.

### Milestone 21.3: Day 9 - All in a Single Night
- **Input:** Distances between cities.
- **Logic:** Traveling Salesperson Problem (TSP).
- **Challenge:** Permutations, Graph traversal.
- **Feature Usage:** `for..of` (iterating cities), `Map` (graph).
